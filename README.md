# AnyList | Lista de productos con NestJS, Postgres SQL, GraphQL y Apollo Server

## Entorno DEV

1. Clonar el proyecto
2. Copiar el `env.example` y renombrar a `.env`
3. Ejecutar

```
yarn install
```

4. Levantar la imagen (Docker Desktop)

```
docker-compose up -d
```

5. Levantar el backend de NestJS

```
yarn start:dev
```

6. Visitar el sitio

```
localhost:3000/graphql
```

7. Ejecutar la mutation "executeSeed", para llenar la base de datos con informaci√≥n
